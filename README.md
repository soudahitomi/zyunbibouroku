### テーマを選んだ理由
* 自分の視点だけでなく人の投稿を通じてタスクや持ち物などを確認するため
* 優先順位をつけ、効率よく動きたいのにそれができずもやもやしないため
* 他人のタスクを見て感化され、達成感を感じるため。その結果、心や人生がより豊かになるため。
